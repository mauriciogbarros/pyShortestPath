# Shortest Path Algorithm
To write a python function that computes the shortest path between the nodes in a graph, and also returns the path taken.

For example, given a graph where cities are connected by roads with different distances, the algorithm will find the shortest route from one city to another. If you want to travel from city A to city D, the algorithm might find that going `A -> B -> C -> D` (total: 15 km) is shorter than going directly `A -> D` (20 km).
